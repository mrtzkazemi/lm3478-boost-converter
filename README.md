# lm3478-boost-converter
2S Li-ion to 12–16V Boost Converter, 1.5A, open source hardware

- Input: 2S Li-ion (6.4–8.4 V)  
- Output: 12–16 V adjustable, 1.5 A  
- Controller: LM3478
